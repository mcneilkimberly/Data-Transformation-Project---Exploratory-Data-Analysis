Performance Trends in the 2025 Connecticut Sun Season

Morde Bernstein, Marley McGehee, Kimberly McNeil

4/17/2026

Introduction

Our dataset is the 2025 schedule and results for the Connecticut Sun, a WNBA team. The data looks at the dates when the games were played, who they played against, the scores, as well as whether they were on a win/lose streak. Additional factors include where they played the game, if it went into overtime, and any additional notes. The sample size is 44. We obtained the data from Basketball Reference’s website on the 2025 Connecticut Sun Schedule and Results page.

Data Import

basketball <- read_csv("connecticut_sun_2025.csv", 
                       col_types = cols(Date = col_date("%a %b %d %Y")))
## New names:
## • `` -> `...3`
## • `` -> `...5`
## • `` -> `...6`
# Initialize cleaning the column names (and giving names to the unnamed)
basketball <- janitor::clean_names(basketball) 

# Make the janitor's work clearer
basketball <- basketball %>%
  rename(
    game = g,
    home_or_away = x3,
    result = x5,
    overtime = x6,
    team_points = tm,
    opp_points = opp,
    total_wins = w,
    total_losses = l
  ) %>%
  mutate(
    home_or_away = ifelse(is.na(home_or_away) | home_or_away == "", "Home", "Away"),
    overtime = ifelse(is.na(overtime) | overtime == "", "False", "True")
)

We first imported the data as a .csv file, then used a function to make the date variable be recognized as date data instead of simply text. While plain text is easy for humans to read, computers can have difficulty, and this will allow it to understand that it is a date and be able to validate our data (e.g. dates like Feburary 30th will be flagged and denied) and perform unique calculations (e.g. timespans). When compared to most datasets with dates, this dataset uniquely contains an abbreviated day of the week in the date variable, but we decided not to include it in our current table because later functions will be able to pull the day of the week from the date alone, so it is not necessary. Following this, we cleaned the names of the columns and updated missing/confusing labels so they would be easier to both use and understand going forward.

Team Score

basketball %>%
  summarize(
    Count = n(),

      Min = min(team_points, na.rm = TRUE),
      Q1 = quantile(team_points, 0.25, na.rm = TRUE),
      Q3 = quantile(team_points, 0.75, na.rm = TRUE),
